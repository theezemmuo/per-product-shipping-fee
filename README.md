Charge per-product shipping fees and disable free shipping automatically.

== Description ==
This plugin allows WooCommerce store owners to assign a flat shipping fee to each product individually. Free Shipping will automatically be disabled if any product has a shipping fee. Perfect for products that require different shipping rates, like solar panels and cables.

Shipping fees are calculated once per product in the cart, ensuring accurate checkout totals. Fully compatible with worldwide shipping.

== Installation ==
1. Upload the `per-product-shipping-fee` folder to the `/wp-content/plugins/` directory, or install via WordPress Admin → Plugins → Add New → Upload Plugin.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Edit a product and enter a shipping fee in the new "Additional Shipping Fee" field.
